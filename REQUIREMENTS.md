# Premium Streaming Platform UI Prompt

## Overview

Create a modern premium movie/video streaming web platform inspired by Netflix, Disney+, Prime Video, and Apple TV+.

The platform should feel cinematic, modern, luxurious, and production-ready.

DO NOT build authentication pages (login/register/forgot password).  
Assume the user is already authenticated and only build the application after login success.

---

# Tech Stack Requirements

Use the following technologies:

- ReactJS
- TailwindCSS
- shadcn/ui
- lucide-react icons
- React Router
- Framer Motion
- Responsive Design
- Mock Data only
- No backend required
